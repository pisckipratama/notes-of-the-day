- install devedencies with ```npm install```
- run REST API with ```npm start```
- open the ```index.html``` file in ```./client``` at your browser or using [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension if you use VS Code.
